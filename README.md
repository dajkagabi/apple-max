#Apple Max

#Telepítés
1. Előfeltételek:
- Node.js (ajánlott a legújabb LTS verzió)
- Git

2.Telepítési lépések:
- Klónozd a tárolót a helyi gépedre:
```
git clone https://github.com/dajkagabi/apple-max.git
cd apple-max
```
3. Telepítsd a függőségeket:
```
npm install
```
4. Indítsd el a fejlesztői szervert:
```
npm run dev
```

A projekt mostantól elérhető a böngésződben a http://localhost:5173/ címen.

5. Funkciók:
- Termékinformációk
- Reszponzív dizájn
- Intuitív navigáció

6. Technológia
Főbb technológiák:
- Vite
- React:
- Tailwind CSS
  
7. Könyvtárak

- @vitejs/plugin-react
- react
- react-dom
- lucide-react
- tailwindcss
- eslint
